# Connections

Connections sind explizite Profile zu externen Systemen.

Aktuell unterstuetzte Familien sind:

- SSH
- SFTP
- SMB
- RSS
- Discord
- HTTP API
- Webhook
- SMTP
- IMAP
- MQTT

Die Routing-Qualitaet steigt, wenn Connection-Metadaten gepflegt sind:

- Titel
- Kurzbeschreibung
- Aliase
- Tags
