# ARIA Wiki

Willkommen im ARIA Wiki.

Aktuelle Public Alpha: `0.1.0-alpha298`.

ARIA ist ein schlanker selbst gehosteter AI-Assistent fuer Menschen, die Kontrolle statt Plattform-Bloat wollen. Die aktuelle Produktlinie ist recipe-first, connection-aware und nutzt LLMs fuer begrenzte Action-Planung, waehrend Guardrails entscheiden, was wirklich laufen darf.

## Aktuelle Produktschwerpunkte

- Memory und Dokument-RAG mit Qdrant
- Notizen als Markdown-Arbeitsbereich
- explizite Connections mit Aliases/Tags fuer Routing
- Rezepte statt Legacy-Skills
- Agentic Action Flow: Kontext, LLM-Draft, Policy/Guardrail, Runtime
- One-Click-Bestaetigung fuer ausgehende Aktionen
- sichtbare Token-/Kosten-Erfassung
- kontrollierte Managed Updates

Dieselben Kerninhalte sind lokal in ARIA unter `/help` verfuegbar.

- [Alpha-Help-System](/help?doc=alpha-help-system) fuer die ausfuehrliche lokale Hilfe
- [Help-System](/help?doc=help-system) fuer Pflegehinweise zur Doku
