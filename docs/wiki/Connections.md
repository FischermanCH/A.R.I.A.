# Connections

Connections are explicit profiles to external systems.

Supported families currently include:

- SSH
- SFTP
- SMB
- RSS
- Discord
- HTTP API
- Webhook
- SMTP
- IMAP
- MQTT

Routing quality improves when connection metadata is maintained:

- title
- short description
- aliases
- tags
